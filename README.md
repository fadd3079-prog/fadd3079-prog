## About

I am an active undergraduate Informatics student at **Universitas Jenderal Soedirman**, Indonesia.  
I focus on building software that is useful, readable, and visually structured.

My current work explores:

- full-stack web applications
- Android apps with offline-first storage
- dashboards and role-based systems
- UI/UX implementation
- database design and access control
- open-source documentation and release workflow
- visual identity and product presentation through **Fadd Graphics**

I keep my projects practical: clear scope, maintainable structure, and interfaces that are easy to understand.
